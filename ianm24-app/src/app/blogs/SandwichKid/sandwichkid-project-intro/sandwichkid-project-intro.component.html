A game about the journey for a perfect sandwich: Sandwich Kid Project Intro

Posted 12 June 2020 4:45PM EST

Updated 12 June 2020 6:40PM EST

So for a few years now, a group of friends and I have wanted to make a game over the summer. With internships cancelled and everyone stuck inside due to COVID-19, we all decided this summer we would actually do it. Our development group has been named Goon Games. The premise of the game is a 2D adventure game where you explore the world to find different sandwich pieces to eventually get you to the perfect sandwich.
